# Trắc Nghiệm CNTT Cơ Bản

Trang web luyện thi trắc nghiệm Kỹ năng sử dụng CNTT cơ bản.

## Tính năng
- 6 mô đun: CNTT cơ bản, Máy tính, Word, Excel, PowerPoint, Internet
- Chọn số câu hỏi và thời gian thi
- Trộn câu hỏi ngẫu nhiên
- Chấm điểm tự động
- Xem lại đáp án chi tiết
- Hỗ trợ phím tắt (A/B/C/D, mũi tên trái/phải)
- Responsive trên mobile

## Deploy
Upload toàn bộ thư mục lên GitHub Pages hoặc bất kỳ static hosting nào.
